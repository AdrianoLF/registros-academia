# Gym System OOP

Sistema de gerenciamento de academia desenvolvido em TypeScript utilizando os conceitos de Programação Orientada a Objetos (POO).

## Sobre o Projeto

O projeto tem como objetivo simular o funcionamento de uma academia, permitindo o gerenciamento de alunos, planos, pagamentos e controle de acesso, aplicando princípios fundamentais de POO como:

- Encapsulamento
- Herança
- Polimorfismo
- Abstração

## Funcionalidades

- Cadastro de alunos
- Cadastro de funcionários
- Gerenciamento de planos
- Controle de mensalidades
- Registro de pagamentos
- Controle de acesso à academia
- Organização de treinos

## Tecnologias Utilizadas

- TypeScript
- Node.js
- Programação Orientada a Objetos (POO)

## Objetivo Acadêmico

Este projeto foi desenvolvido com fins acadêmicos para praticar conceitos de Programação Orientada a Objetos e organização de sistemas utilizando TypeScript.
